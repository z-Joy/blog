    微信小程序node环境

    如果你的微信小程序项目需要依赖node环境，那么可能会遇到下面提到的问题！

    开发者工具中提示/bin/sh: npm: command not found

    在使用nvm版本管理器安装node的情况下，一般会出现这种情况。

    解决方法是找到node的安装目录，将其链接到 /usr/local/bin/ 目录下面。

    • 1、使用 which node 命令找到node的安装目录
    • 2、使用 sudo ln -s /Users/xxx/.nvm/versions/node/[版本号]/bin/node /usr/local/bin/node 命令建立链接

    npm包重复上述的1、2步骤完成链接建立。

    现在在你的微信开发者工具中就可以正常使用node环境开发了。
